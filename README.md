# main-app-hub

## 项目简介

`main-app-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，旨在提供一个通用的页面集合，方便开发者或团队集中管理、共享与测试各类前端页面。

## 目录说明

```
main-app-hub/
├── index.html          # 仓库入口页面，列出所有已归档页面
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 静态资源目录（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个独立 HTML 页面作为一个文件存放于此，文件名应具描述性。
- `assets/`：存放各页面共用的样式、脚本或图片资源，避免重复引入。
- `index.html`：可通过浏览器直接访问，展示所有归档页面的链接列表。

## 页面归档说明

1. **添加新页面**：在 `pages/` 目录下创建新的 HTML 文件，并更新 `index.html` 中的链接列表。
2. **资源引用**：如需引用外部资源，建议优先使用相对路径或统一放置于 `assets/` 目录。
3. **命名规则**：文件名应简洁且能反映页面内容，避免使用特殊字符或空格。
4. **独立性**：每个页面应尽量保持独立，不依赖其他页面或外部服务。
